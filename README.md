# Habit Tracker
## A Simple Habit Tracking Cross-Platform App

This application includes...
- a Homepage
- a Analytics Page
- a Empty State Page
- a Habit Statistics & Data Analytics
- a Habit Editing & Creation Bottom Sheet
- and other UI pages and elements.
